For this branch. 
You need to download / install the following
1. Docker
2. Node
3. Angular
4. postgreSQL
5. Postman
6. Java Springboot
7. Maven installed
8. Other related VSC extensions
- Dev Containers
- Docker
- Docker Compose
- Docker Explorer
- GitLens
- Maven for Java
- Prettier - Code formatter
- Red Hat Dependency Analytics


1. When in doubt, run this -> 
1. cd backend
2. mvn clean
3. mvn install
4. mvn compile
5. mvn spring-boot:run
6. or all in one, mvn clean install compile spring-boot:run

7. cd frontend
8. npm install
9. ng serve -o
