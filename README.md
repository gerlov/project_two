For this branch. 
You need to download / install the following
1. Docker
2. Node
3. Angular
4. postgreSQL
5. Postman
6. Java Springboot
7. Maven installed
8. Other related VSC extensions
- Dev Containers
- Docker
- Docker Compose
- Docker Explorer
- GitLens
- Maven for Java
- Prettier - Code formatter
- Red Hat Dependency Analytics


When in doubt, run this -> 
cd backend 
mvn clean
mvn install
mvn compile 
mvn spring-boot:run

cd frontend
npm install
ng serve -o
